## Prerequisites

Before installing citrouille, ensure you have:

### Required

- **Python 3.12 or higher**
  ```bash
  $ python --version
  # Should show Python 3.12.x or higher
  ```

- **pip** (Python package installer)
  ```bash
  $ pip --version
  ```



## Installation Methods

### Install from upstream (Recommended)

The easiest way to install citrouille is via pip from PyPI:

```bash
pip install citrouille
```

### Install from Source

Please refer to the [development guide](./development.md)

### Method 3: Install from GitHub Release

Download and install a specific release:

```bash
pip install https://github.com/Chelsea486MHz/citrouille/releases/download/v1.0.0/citrouille-1.0.0-py3-none-any.whl
```
